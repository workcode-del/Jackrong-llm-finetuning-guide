## 🧭 How to Add a New Guide

When adding a new PDF or report:

| Field | Expected content |
|---|---|
| File | Store the PDF directly in `guidePDF/` with a descriptive filename. |
| Topic | Summarize the main workflow, model, dataset, or evaluation focus. |
| Audience | Identify whether the guide is beginner-facing, developer-facing, or research/technical. |
| Language | State the primary language or multilingual status. |
| Links | Update this catalog first, then add a compact summary to the root README only when it is broadly useful. |
